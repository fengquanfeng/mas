## 目录说明

| 目录 | 说明 |
|------|------|
| `api/` | 统一封装后端 API 接口，按模块组织 |
| `assets/` | 静态资源，包括图标、图片等 |
| `components/` | 全局可复用组件，以 `sc` 前缀命名 |
| `config/` | 项目配置文件 |
| `directives/` | Vue 自定义指令 |
| `layout/` | 页面布局组件 |
| `locales/` | 国际化多语言配置 |
| `router/` | Vue Router 路由配置 |
| `store/` | Vuex 状态管理 |
| `style/` | 全局样式文件 |
| `utils/` | 工具函数库 |
| `views/` | 页面视图组件 |

## 主要文件说明

| 文件 | 说明 |
|------|------|
| `main.js` | 应用入口，初始化 Vue 实例 |
| `App.vue` | 根组件 |
| `scui.js` | SCUI 核心，注册全局组件和属性 |
